# Shreyaj Padigala - Portfolio Website

A modern, responsive portfolio website built with Next.js and Tailwind CSS, showcasing my projects, experience, and skills as a Data Science student and Software Engineer.

## Features

- **Modern Design**: Clean, professional design with smooth animations using Framer Motion
- **Responsive**: Fully responsive layout that works on all devices
- **Dark Theme**: Beautiful dark theme optimized for readability
- **Fast Performance**: Built with Next.js for optimal performance and SEO
- **Smooth Animations**: Engaging animations and transitions throughout the site
- **Interactive Components**: Hover effects, scroll animations, and interactive elements

## Tech Stack

- **Framework**: Next.js 16
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Language**: TypeScript
- **Icons**: React Icons
- **Deployment**: GitHub Pages

## Sections

- **Hero**: Introduction with social links
- **About**: Education, coursework, and activities
- **Experience**: Professional and research experience
- **Projects**: Featured projects with descriptions and technologies
- **Skills**: Technical skills organized by category
- **Contact**: Contact information and social links

## Development

To run this project locally:

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions. Any push to the main branch triggers a new deployment.
